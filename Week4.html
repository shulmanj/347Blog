CSS Triangle

This week I decided to learn how to make a simple shape using only CSS: a triangle. To be honest, I'm not entirely sure what this could be used for. Maybe the roof of a house or a direction on a compass? Anyway, the code itself is super simple: 

.up {
    width: 0;
    height: 0;
    border-left: 70px solid transparent;
    border-right: 70px solid transparent;
    border-bottom: 70px solid cornflowerblue;
}

This CSS creates a triangle pointing up. Essentially, all it is is a box with a width and height of 0. The size of the triangle is determined by the thickness of the border. In an up facing triangle, the left and right borders are the same size as the bottom border, but their colors are transparent. This allows the triangle shape to still appear.

If the other borders were not set to transparent, the result would be rectangles and trapezoids:

Although it may not be entirely useful, it demonstrates that you can make a few simple shapes with CSS alone, not using SVG.
