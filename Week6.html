Changing Cursor With JavaScript

Another week, another blog post. This week, I wanted to look further into what all I can do using relatively simple JavaScript along with CSS.

As you can see, whenever you select a new cursor, it is reflected in the document. This is done fairly simply using JavaScript and some buttons: 

<select id="new-cursor">
        <option>default</option>
        <option>crosshair</option>
        <option>e-resize</option>
        <option>help</option>
        <option>move</option>
        <option>n-resize</option>
        <option>ne-resize</option>
        <option>nw-resize</option>
        <option>pointer</option>
        <option>progress</option>
        <option>s-resize</option>
        <option>se-resize</option>
        <option>sw-resize</option>
        <option>text</option>
        <option>wait</option>
</select>

This is dropdown menu allows the user to select a new cursor.

<input type="button" onclick="example_change_cursor()" 
        value="Change Cursor"/>
<input type="button" onclick="example_restore_default()" 
        value="Restore Default Cursor"/>

These buttons link to the JavaScript functions that actually change the cursor: 

<script>
        function example_change_cursor() {
            document.body.style.cursor = 
                document.getElementById('new-cursor')
                .options[document.getElementById('new-cursor')
                .selectedIndex].value;
        }

        function example_restore_cursor() {
            document.body.style.cursor = 'default';
        }
    </script>

These functions take the name of the cursor from the dropdown menu and then apply that to the page.

In my opinion, I don't think that allowing the user to change their cursor adds any benefit to the design of a webpage more than just a bit of fun. It is way more functional to keep the system's default.
