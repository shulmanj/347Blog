JavaScript Animation

This week, I decided to look into animations using JavaScript as opposed to CSS alone. This relatively simple animation moves a small box diagonally across a larger box.

The two boxes are created by using two divs and setting their width, height, position, and colors: 

#big-box {
    width: 400px;
    height: 400px;
    position: relative;
    background: rgb(69, 0, 132);
}
#little-box {
    width: 50px;
    height: 50px;
    position: absolute;
    background-color: rgb(203, 182, 119);
}

I then created a button. When that button is pressed, it calls the function animateBox. 

<p id="button">
    <button onclick="animateBox()">Animate!</button>
</p>

When this button calls animateBox, it triggers the animation. The JavaScript gets the size and color information from the smaller, gold box, sets its relative position to 0, and until it reaches position 350, which with a width and height of 50px, would be the corner of the larger box, moves the gold box. Once it reaches position 350, it stops the animation. 

function animateBox() {
    let id = null;
    const elem = document.getElementById("little-box");
    let pos = 0;
    clearInterval(id);
    id = setInterval(frame, 5);
    function frame() {
        if (pos == 350) {
            clearInterval(id);
        }
        else {
            pos++;
            elem.style.top = pos + "px";
            elem.style.left = pos + "px";
        }
    }
}

This easy animation gave me more of an understanding of how JavaScript can be embedded directly into HTML as well as how powerful it can be.
